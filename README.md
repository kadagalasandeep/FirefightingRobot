🔥 Fire Fighting Robot using IoT

🚀 Overview

This project is an autonomous fire fighting robot that detects fire using flame sensors and extinguishes it using a servo-controlled water pump system.

It is designed as a smart IoT-based safety system for homes, industries, and hazardous environments.

---

⚙️ Features

- 🔥 Automatic fire detection using flame sensors
- 🎯 Direction-based targeting (Left, Center, Right)
- 🤖 Servo motor controlled water spraying
- 💧 Water pump activation using relay
- 📡 IoT-ready (ESP32 integration possible)

---

🧠 How It Works

1. Flame sensors detect fire in different directions
2. Arduino processes sensor data
3. Servo rotates towards fire direction
4. Water pump activates to extinguish fire

---

🧰 Components Used

- Arduino UNO / ESP32
- Flame Sensors (3x)
- Servo Motor (SG90)
- Relay Module
- Water Pump / Fan
- Battery Pack
